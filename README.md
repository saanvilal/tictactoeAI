# tictactoeAI

A Python-based Tic-Tac-Toe game where players can compete against another player or an AI opponent. This project was built to explore game logic and AI using the minimax algorithm with alpha-beta pruning.

## Features

- Player vs AI mode
- Player vs Player mode
- Difficulty levels: Easy, Medium, Hard
- Minimax algorithm with alpha-beta pruning
- Win, draw, and quit handling through the command line

## Technologies Used

- Python
- Minimax algorithm
- Alpha-beta pruning

## How It Works

The game allows users to choose between playing another person or against an AI. In AI mode, the computer evaluates possible moves using the minimax algorithm and improves decision-making based on the selected difficulty level. Alpha-beta pruning is used to make the search more efficient.
